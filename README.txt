Para la resolución de este trabajo práctico deberán trabajar en grupos de dos personas y resolver las siguientes consignas:

1- Clonar el siguiente repositorio: https://github.com/nzalazar/tprecuperatorio.git
2- Cada integrante deberá del corregir los errores de código presentes en los archivos: Búsqueda secuencial y Ordenamiento burbuja para que el script php funcione correctamente. (un script cada uno)
3- Deberán utilizar los comandos de git para configurar el repositorio, agregar los cambios y confirmar (cada uno con su usuario correspondiente)
4- Deberán enviar por correo la carpeta comprimida del repositorio (uno por grupo) y la captura de pantalla de la salida del comando git log donde se puedan observar los cambios realizados por cada uno de los usuarios.
